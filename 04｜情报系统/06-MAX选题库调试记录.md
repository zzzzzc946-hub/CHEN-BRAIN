# MAX选题库调试记录

## 文件定位

本文件用于记录 MAX 选题库在真实使用前的调试过程。

它不是外部情报日报。

它不是选题库本体。

它不是内容生产 Skill。

它记录的是：

```text
一条外部情报 / 候选材料 / 选题卡片，
在给 MAX 口喷之前，
经过 CHEN / MAX / AI 讨论后，
具体调试了什么，
为什么要调试，
调试后变成了什么样，
以及这次调试应当反向更新哪些规则。
```

本文件的主作用是：

```text
校准 MAX选题库里“给 MAX 口喷的材料卡片”应该长什么样。
```

同时，它也会反向影响：

```text
外部情报日报怎么收集
日报里的“初步 MAX 可讲视角”怎么写
关键词与来源池应该补充哪些检索方向
选题规则应该如何更新
```

一句话：

```text
这里记录的不是素材本身，而是素材被调试成 MAX 可讲材料的过程。
```

---

# 一、记录原则

每一次调试，都必须记录清楚四件事：

```text
1. 原来是什么
2. 哪里不对
3. 怎么调对
4. 调试后要更新哪些规则
```

调试记录不能只写结论。

必须保留调试过程。

因为后续系统会越来越准，靠的不是一次性写对，而是反复记录：

```text
AI 最初怎么理解
CHEN / MAX 怎么纠偏
纠偏后产生了什么更高质量的判断
这个判断是否要沉淀成长期规则
```

# 2026-06-16｜约会香水案例的切口能量调试

## 1. 调试对象

约会香水案例。

这条材料原本来自 CHEN 对具体约会细节的构造：

```text
一个男生约会前精心准备了穿搭、餐厅、展示面，甚至想用香水增加精致感。
但他平时很少真正使用香水，也不懂品牌、浓度、场合和用量。
结果约会时香水味太冲，或者女生问他喷的什么，他一说出来是杂牌、盗版、廉价网红香。
前面所有准备瞬间破掉。
```

## 2. 原始材料

原始判断是：

```text
香水本来是加分项，但如果使用不自然，可能变成身份破绽。
懂的女生会判断：这个人不是长期有这种生活习惯，而是在临时补一个“精致男人”的人设。
```

这个素材的价值在于：

```text
它是一个极小、具体、可感的约会场景。
它能反映价值感、生活方式熟悉度、审美细节和身份识别。
```

## 3. AI 初始处理

AI 初始给出的推荐切口是：

```text
你用的香水，暴露的不是价格，是你根本不熟这种生活。
```

AI 当时的理解偏向：

```text
香水是一个细节破绽。
错误香水会暴露生活方式不熟。
展示面可能被一个小细节打穿。
```

## 4. CHEN / MAX 反馈

MAX 反馈：

```text
这个切入点已经有 70 分，可以讲。
但“香水暴露你不熟这种生活”这个视角稍微偏负，甚至有点偏穷人视角。
```

MAX 更认可的方向是：

```text
不要从指责、审判、羞辱进入。
更好的方式是像 JZ 一样，提供知识、品味、生活方式和选择建议。
```

MAX 提出的更好方向：

```text
这更像是“约会香水选择”。
一个细节就可以让女生快速上头。
这个角度能量更好。
```

CHEN 补充判断：

```text
重点不是一定要讲“你被看穿了”。
重点是这个具体香水场景能引发 MAX 的观点。
这个素材搜集就是成功的，因为它触发出了更准确的内容方向。
```

## 5. 问题诊断

这次偏差属于：

```text
切口能量不对。
问题诊断成立，但表达容易偏负。
容易从“价值提供”滑向“细节审判”。
容易像传统情感号一样制造压迫感。
```

具体问题：

```text
“你根本不熟这种生活”这句话虽然锋利，但容易给人居高临下、批判、羞辱的感觉。
它更像在指出用户的低位，而不是带用户进入一个更好的选择。
```

MAX 更想要的表达气质：

```text
没有戾气。
不居高临下。
不批判穷人。
不压迫用户。
像 JZ 一样，用知识科普、品味判断和生活方式启发去提供价值。
```

## 6. 调试后的新方向

调试后，这条材料不应主要讲：

```text
你喷错香水，会暴露你不熟这种生活。
```

而应更适合讲：

```text
约会香水选择，是一个能让女生快速感受到品味、干净感、距离感和生活细节的入口。
```

它的方向从：

```text
负向破绽识别
```

调整为：

```text
正向品味建议 / 约会细节价值提供 / 生活方式启发
```

## 7. 调试后的可用表达

可用表达 1：

```text
约会时，一个香水细节就能让女生快速感受到你的品味。
```

可用表达 2：

```text
约会香水不是为了让人闻到你喷了香水，而是让她觉得你这个人很干净、很有距离感。
```

可用表达 3：

```text
真正加分的香水，不是贵，而是符合你的气质、场合和距离。
```

可用表达 4：

```text
一个男人的精致感，不是靠香水堆出来的，而是靠他知道什么场合该用什么味道。
```

保留为底层判断的表达：

```text
你用的香水，暴露的不是价格，是你是否真的熟悉这种生活。
```

注意：

```text
这句话可以作为底层判断，但不一定适合作为最终外部切口。
```

## 8. 沉淀规则

本次调试沉淀出以下规则：

```text
外部情报可以从翻车、尴尬、破绽进入。
但 MAX 最终表达不能停留在羞辱、批判、低位审判。
```

进一步规则：

```text
抓取可以抓破绽。
日报可以记录破绽。
选题库给 MAX 的材料，必须尽量转成正向价值、知识启发、品味判断和场景建议。
```

切口优先级：

```text
正向价值提供
↓
知识 / 品味 / 生活方式启发
↓
具体场景建议
↓
问题诊断
↓
刺痛 / 羞辱 / 批判
```

这条规则尤其适用于：

```text
约会细节
展示面破绽
审美误判
生活方式不熟
情感关系低位
```
