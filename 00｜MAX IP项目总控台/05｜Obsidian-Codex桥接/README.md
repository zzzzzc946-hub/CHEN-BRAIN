# Obsidian-Codex 桥接

## 用法

在 Obsidian 当前笔记中运行 Templater 模板：

```text
Codex｜处理当前笔记
```

输入你要 Codex 做的事。

Codex 会读取当前笔记全文，并把结果追加到当前笔记末尾。

## 当前能力

- 基于当前 Obsidian 笔记提问
- 基于当前笔记整理、总结、提炼
- 按指令局部修改 CHEN BRAIN 文件
- 输出结果回写到当前笔记

## 边界

- 不主动 commit / push
- 不主动扩系统
- 不默认修改 `02｜MAX顶层思维系统`
- 长任务可能需要等待几十秒到数分钟
