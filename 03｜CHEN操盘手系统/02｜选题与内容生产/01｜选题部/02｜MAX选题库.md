# 02｜MAX选题库

## 0｜文档定位

本文档用于承接 CHEN 和 AI 共创出来的选题前端结果，也用于记录 MAX 口喷、内容成型后反推出来的选题和标题。

它不只接收最终标题和成品选题。

前半部分记录：

```text
话题 / 母题
具体场景 / 现象
现象带来的结果 / 反馈 / 争议
多个切入点
推荐切入点
```

后半部分在 MAX 口喷、内容成型后补充：

```text
反推选题
反推标题
成稿链接 / 内容记录
发布状态
```

不要把没有经过内容生产的标题当成最终标题。

标题和选题必须允许后置反推。
