```text
目标用户为什么会关心：
MAX 用户往往积累了不少好照片：酒店、车、工作、旅行和健身单独看都不错，放到一起却像不同人。每张都在证明一项优势，整组没有节奏，也没有统一身份。

真正高级的朋友圈不是每张都必须成为大片，而是知道哪张负责人物、哪张负责环境、哪张提供信息、哪张负责留白。审美不仅是会拍，也包括会选、会排和会删。

MAX 可讲切入点：
1. 为什么每张都不错，朋友圈还是不好看。
2. 九宫格为什么不是九张照片，而是一套视觉叙事。
3. 每张都想证明自己，为什么整组会很用力。
4. 风格统一和画面重复有什么区别。
5. 审美为什么体现在删掉什么。

推荐切入点：
单张照片好看是摄影，九张照片放在一起还像同一个人，才叫审美。
```

```text
高端展示面 / 第一判断 / 陌生人信任 / 信息传递 / 长期杠杆

多条素材共同指向展示面的基础功能：女生认识一个陌生男人后，往往会先看主页。她未必先计算这个男人有多少钱，而是确认他是谁、生活是否正常、状态是否稳定、有没有明显风险，以及是否值得继续了解。

展示面不是照片合集，也不是单纯引流。它是一套帮助陌生人降低理解成本的信息系统。话术和线下行动只能承接已经形成的初步印象，无法替代身份、生活和状态的基础信息。

经营展示面不是讨好别人，而是对自己的真实价值负责。它既负责吸引，也负责筛选；既让适合的人更快理解你，也让不匹配的人更早离开。

MAX 可讲切入点：
1. 很多关系为什么不是聊崩的，而是点开主页就结束了。
2. 女生看朋友圈第一步为什么是在找风险信号。
3. 话术为什么救不了空白朋友圈。
4. 展示面为什么是长期杠杆，而不是面子工程。
5. 展示面和虚假包装的边界。

朋友圈首先要让别人看懂你、放心继续了解你，然后才轮得到心动。
```

```text
一条搞笑丑照可能得到很多朋友评论，一条工作吐槽可能引发大量共鸣，一张车表照片也可能收获点赞，但陌生女生没有熟人上下文。她会把每条内容当作证据，推测这个男人的生活、情绪、审美和相处状态。

4. 朋友圈为什么是一套人格证据，而不是内容清单。
5. 同一种内容为什么有人加分、有人减分。

朋友圈最怕的不是没人点赞，而是点赞很多，女生看完还是不想认识你。
```
