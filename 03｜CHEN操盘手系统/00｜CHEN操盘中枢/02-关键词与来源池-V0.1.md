# 关键词与来源池 V0.1

## 文件定位

本文件承接《01-全网情报检索规则》。

它负责回答：

```text
AI / 后续 Agent / CHEN 在检索外部情报时，
应该从哪些平台、热榜、关键词、账号、来源入口开始找。
```

本文件不是判断规则。

素材是否进入《MAX外部情报日报》，以《01-全网情报检索规则》为准。

本文件不是日报。

每日或每轮检索结果进入《MAX外部情报日报》。

本文件不是选题库。

只有经过《选题规则》二次处理的材料，才进入《MAX选题库》。

本文件不是技术采集方案。

爬虫、OCR、ASR、评论抓取、链接解析、账号监控，后续进入《平台采集执行方案》。

# 一、核心原则

## 1. 关键词池不是边界

关键词池只是高概率入口，不是搜索边界。

任何未被关键词覆盖的热点、新闻、人物事件、平台热议，只要符合《01-全网情报检索规则》，都可以进入候选情报。

本系统有三种入口：

```text
开放热点雷达：捕捉突发热点、新闻、热榜和平台热议。
稳定关键词池：主动检索长期有效的用户痛点和现实场景。
来源池：长期观察高价值账号、话题、栏目和信息源。
```

一句话：

```text
关键词池负责主动找。
开放雷达负责捕捉意外热点。
来源池负责长期盯高价值入口。
01 检索规则负责判断是否进入日报。
```

## 2. 不找观点，找现实暴露点

优先找：

```text
具体事件
具体动作
具体场景
具体现象
具体关系反馈
具体社会争议
具体评论区共识
```

不优先找：

```text
价值感
高价值男人
吸引力逻辑
展示面重要性
两性认知
男人框架
```

原因：

```text
概念词容易搜到别人已经讲完的说教内容。
现实暴露点才容易被 MAX 重新解释。
```

## 3. 所有检索必须回到 MAX 用户痛点

MAX 当前阶段不是对所有男人说话。

核心用户是：

```text
有真实价值，但没有被正确看见的高价值男性。
```

所以外部情报优先服务这些问题：

```text
真实价值为什么没有被感知？
为什么有价值但没有价值感？
为什么条件不错但女生没感觉？
为什么展示面没有表达出真实段位？
为什么拍得贵但还是显得土？
为什么有钱但没有吸引力？
为什么在关系里把价值表达成供养、讨好和低位？
为什么高分女生没有被打动？
为什么普通展示面团队拍不出高端感？
```

# 二、平台检索入口

以下是各平台的高概率优势方向，不是排他规则。

任何平台只要出现具体事件、具体动作、具体现象、社会规律、情感案例、审美误判、名利场规则，都可以采集。

## 1. 小红书

高概率优势方向：

```text
女性视角
约会细节
朋友圈评价
展示面反馈
穿搭审美
生活方式评价
评论区真实语言
```

重点观察：

```text
女生通过什么细节判断一个男人不熟、不自然、太用力、没感觉。
```

## 2. 抖音

高概率优势方向：

```text
爆款现实判断
同行话题
情感案例
阶层审美
热点延伸
评论区共识
高传播表达
```

重点观察：

```text
哪些现实判断正在被大量传播，哪些话题形成了公共情绪。
```

## 3. 视频号

高概率优势方向：

```text
中年男性处境
供养者模式
家庭关系
现实情感困境
熟人社会里的价值消耗
传统关系里的男性压力
```

重点观察：

```text
一个男人的价值如何在现实关系里被消耗、误读或表达成低位。
```

## 4. 微博

高概率优势方向：

```text
热搜
明星绯闻
公共争议
文娱热点
社会情绪
快速发酵事件
评论区舆论
```

重点观察：

```text
热点背后有没有身份差异、审美判断、关系结果或场域规则。
```

## 5. B站

高概率优势方向：

```text
长视频分析
影视片段
审美进阶
阶层文化
生活方式分析
情感长案例
```

重点观察：

```text
能不能从长内容中拆出具体场景、具体动作和稳定规律。
```

## 6. 知乎

高概率优势方向：

```text
真实故事
匿名经历
两性关系
原生家庭
人格模式
择偶观
社会规律
```

重点观察：

```text
具体故事里有没有稳定关系模式。
匿名故事可以作为典型场景，不默认当成事实事件。
```

## 7. 微信公众号

高概率优势方向：

```text
深度文章
品牌活动
名利场观察
人物故事
商业趋势
生活方式趋势
时尚与奢侈品观察
```

重点观察：

```text
短视频里只有结论的东西，公众号可能能补充背景、人物和事件细节。
```

# 三、开放热点雷达

开放热点雷达用于捕捉关键词池无法提前覆盖的突发事件。

它的规则是：

```text
热点不需要提前存在于关键词池。
只要它能被 MAX 重解释，就可以进入候选情报。
```

## 1. 开放雷达入口

重点观察：

```text
抖音热榜
微博热搜
微博文娱热搜
小红书热议
视频号推荐流 / 热门内容
B站热门 / 排行榜
知乎热榜
公众号热文
新闻平台热榜
体育赛事热议
时尚 / 奢侈品新闻
品牌活动新闻
```

## 2. 开放雷达捕捉对象

```text
突发社会事件
高热情感事件
文娱热点
明星恋情 / 分手 / 争议
体育赛事中的阶层差异
品牌活动翻车
名流 / 富豪 / 家族事件
两性关系争议
公众人物审美翻车
社会价值观争议
平台高赞争议内容
```

## 3. 热点进入候选的判断

热点不因为热就进入日报。

必须能回答至少一个问题：

```text
它是否暴露了价值表达问题？
它是否暴露了关系模式？
它是否暴露了身份差异？
它是否暴露了审美判断？
它是否暴露了阶层 / 场域规则？
它是否能解释目标用户为什么没有被正确看见？
```

## 4. 示例说明

类似高热情感事件，不一定会提前出现在关键词池里。

它应该通过开放雷达捕捉：

```text
微博热搜
抖音热榜
知乎热榜
平台热议
新闻报道
```

再由《01-全网情报检索规则》判断：

```text
它是不是具体事件？
有没有社会反馈？
有没有情感关系、供养、价值表达、关系结果？
MAX 能不能重解释？
```

如果可以，就进入候选情报。

# 五、动态热点词池

动态热点词池用于临时记录某一阶段正在发酵的热点词。

它不是长期关键词池。

热点过期后，可以：

```text
删除
沉淀成长期关键词
沉淀成社会现象
沉淀成案例素材
```

记录格式：

```text
热点词：
来源平台：
事件概述：
为什么值得观察：
可能关联的 MAX 主线：
当前状态：发酵中 / 已过热 / 可沉淀 / 删除
```

示例：

```text
热点词：某高热情感事件
来源平台：微博 / 抖音 / 知乎
事件概述：一段关系中的供养、付出、分手和极端结果引发社会讨论
为什么值得观察：涉及供养者、关系低位、价值表达、男性情感处境
可能关联的 MAX 主线：真实价值如何被错误表达成低位和供养
当前状态：可沉淀为情感关系案例
```

# 六、来源池

来源池不属于日报，也不属于选题库。

它只属于本文件。

它的作用是：

```text
记录哪些账号、热榜、话题、栏目、内容源值得长期观察。
```

后续 Agent 自动化时，来源池可以升级成：

```text
账号监控名单
热榜监控入口
内容源优先级
```

# 七、低效关键词黑名单

以下词不是不能用，但默认低效。

除非和具体场景、动作、结果组合，否则不优先搜。

```text
价值感
吸引力
展示面
高价值男人
两性认知
男人框架
恋爱技巧
情绪价值
如何提升魅力
如何变高级
什么是老钱风
如何穿出高级感
```

低效原因：

```text
这些词容易搜到已经被别人说教过的结论，而不是现实里发生的具体素材。
```

正确用法：

```text
不要搜：展示面
要搜：女生吐槽男生朋友圈太装

不要搜：吸引力
要搜：人不错但没感觉原因

不要搜：老钱风
要搜：老钱风翻车有钱但土
```

# 八、使用方式

## 1. CHEN 临时发现素材

CHEN 看到有意思的视频、案例、热点、新闻、评论区，可以直接发给 AI。

不需要按固定流程手动检索。

AI 收到素材后，按《01-全网情报检索规则》判断：

```text
KEEP / BUFFER / DROP
```

然后决定是否进入：

```text
《MAX外部情报日报》
```

## 2. AI 或后续 Agent 主动检索

AI 或后续 Agent 主动检索时，使用本文档作为入口。

优先顺序：

```text
开放热点雷达
↓
稳定关键词池
↓
已验证来源池
↓
低效关键词过滤
↓
按 01 规则判断是否进入日报
```

这里不写具体自动化技术。

技术执行进入《平台采集执行方案》。

## 3. 动态维护

本文件不设固定每周任务。

维护方式是：

```text
讨论中发现有效关键词，就补进稳定关键词池。
讨论中发现阶段性热点，就放进动态热点词池。
发现高价值账号，就放进来源池。
发现低效关键词，就放进黑名单。
发现新的高价值场域，就补进固定场域来源。
```

后续 Agent 开发时，可以升级成：

```text
关键词命中率统计
账号命中率统计
热点词生命周期管理
低效词自动降权
高价值来源自动升权
```

但 V0 阶段不写成自动化任务。
