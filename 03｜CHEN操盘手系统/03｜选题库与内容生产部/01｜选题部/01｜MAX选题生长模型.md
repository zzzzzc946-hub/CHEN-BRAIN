# 01｜MAX选题生长模型

## 一、选题生长链路

    IP母题
    → 内容母题候选池
    → 话题方向
    → 子话题
    → 视角
    → 具体选题
    → 选题评分
    → 优先测试
    → 脚本
    → 发布
    → 数据复盘
    → 反向迭代
