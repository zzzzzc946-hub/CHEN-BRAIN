# 02｜MAX选题库

## 0｜文档定位

MAX选题库不是单纯标题库，而是承接“全网信息检索 Skill → 选题规则 → 10-MAX内容生产 Skill”的中间沉淀库。

它既记录前端筛选出来的选题前置卡，也记录内容成型后反推出来的最终选题和标题。

选题库不要求一开始就有完整标题。

前端阶段优先记录：

```text
信息来源
话题 / 母题
具体场景 / 现象
结果 / 反馈 / 争议
目标用户为什么会关心
多个切入点
推荐切入点
可供 MAX 口喷的问题
状态
```

内容成型后再补充：

```text
反推选题
反推标题
成稿链接 / 内容记录
发布状态
数据反馈
```
