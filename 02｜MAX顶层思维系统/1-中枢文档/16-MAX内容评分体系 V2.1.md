

**性质：内容生产、发布决策与市场校准的统一标尺** **适用范围：所有MAX外部内容（口播视频为主）** **分制：全体系统一10分制** **与其他文档关系：效力低于《阶段作战纲领》。V1.1的否决门、增量检验、差值校准机制全部继承。**

# 第一部分：体系总览

整套体系分三层，按时间先后发生：

| 层   | 名称    | 时点           | 谁打          | 作用            |
| --- | ----- | ------------ | ----------- | ------------- |
| 第一层 | 事前决策分 | 稿子完成当下       | AI自评 + 人工评  | 决定这条内容毙、改、还是拍 |
| 第二层 | 互动数据分 | 发布后T+2 / T+4 | codex抓取自动计算 | 衡量单条内容的市场接受度  |
| 第三层 | 客资结果分 | 每周日结算        | 人工统计        | 衡量本周内容组合的商业结果 |

**核心原则一：事前分管发布，事后分管校准。** 拍不拍只看事前分，不等市场数据。市场数据用于每周复盘，校准的是选题方向、内容配比和评分标准本身。

**核心原则二：三层分数不合成总分。** 体系的产出不是一个最终数字，而是层与层之间的差值。差值才是信号（详见第五部分）。

**核心原则三：市场分永不覆盖否决门。** 任何红线内容，客资再多也是0分。低段位爆款是负资产——它带来的客资质量低，且损耗MAX的高客单定位。

# 第二部分：否决门（继承V1.1，先于一切评分）

以下任何一条出现，内容直接0分退回，不进入评分：

1. 出现《阶段作战纲领》红线词表禁用词。
2. 出现领域流体系词汇。
3. 出现校准库中明确标注"不得再出现""外部禁用""AI化表述"的条目。
4. 油腻、低俗、侮辱女性，或触碰定性安全。
5. 伪概念堆砌。

## 2.2 慎用词（V2.1新增：可以用，但空用即扣分）

校准库中标注"不要空用""不要滥用"的词（如：不稳、接住、撑住、属于、高级感、贵、价值感等），不禁止使用，但每次出现必须有具体场景、动作或心理支撑。空用（直接贴标签、无支撑下结论）在真实感和MAX贴合度两个维度扣分；通篇多处空用，按退回处理。

**原则：校准库不是禁词表。把校准库当禁词表执行，AI会越来越保守，产出无菌但无味的内容。**

# 第三部分：事前决策分（第一层）

## 3.1 六个维度与权重

|维度|权重|一句话定义|
|---|---|---|
|一、第一印象|15%|不分析，凭直觉：读完有没有被抓住|
|二、真实感|15%|像人说话还是像AI写作|
|三、文章深度|20%|有没有增量——说出别人说不出的那一层|
|四、场景度|15%|是否从目标用户的真实场景和动作出发|
|五、MAX贴合度|15%|像不像MAX的话、MAX的判断、MAX的立场|
|六、用户价值|20%|用户看完带走了什么——痛了、悟了、学到了|

**权重设计逻辑**：深度+用户价值合计40%，对应V1.1最重要的沉淀——增量是内容的生命线，形式分不能补内容分。其余四个维度是底盘，做不好一票拉垮，做好了不单独成就一篇内容。

**事前分 = Σ（维度分 × 权重），满分10分。**

## 3.2 各维度评分锚点

### 维度一：第一印象（15%）

读完第一遍、开始分析之前的直觉。这个维度必须先打，打完再做其他维度的理性分析，防止理性分析污染直觉。

- **10分**：有生理反应——中途停下来回读某一句，读完想立刻转给某个人，第一直觉"这条会爆"。
- **8分**：整体被抓住，有一两处明显的"诶？"，愿意马上读第二遍。
- **6分**：顺畅读完，不反感，但放下手机就忘了。属于"还行"。
- **3分**：读到一半开始扫读、跳行。
- **1分**：第一段就想关掉。

### 维度二：真实感（15%）

- **10分**：完全像MAX本人在镜头前说话。有口语毛边、有停顿、敢断敢停，挑不出一句"写出来的句子"。
- **8分**：整体像人，但有一两处工整对仗或总结式收尾暴露了写作痕迹。
- **6分**：半人半文——判断像人，结构像作文：排比连发、"你以为/其实"式转折、漂亮收尾。
- **3分**：通篇书面语，概念词堆叠，金句一句接一句。
- **1分**：一眼AI，模板结构。

### 维度三：文章深度（20%）

**硬性前置——增量检验**：评分前先回答"用户看完带走了哪一个他原来不知道的判断？"一句话答不出，本维度直接≤3分。常见选题用常见角度切入，本维度直接≤3分。

- **10分**：至少一个判断让人愣住——"我从没这么想过，但他是对的"。这个判断从任何公共素材里都抄不到。
- **8分**：有明确增量，机制讲透或视角确实新，但最深那层在行业里能找到相似表述。
- **6分**：逻辑成立、机制有讲，但核心判断是用户已知道理的精致复述——"说了像没说"。
- **3分**：只有结论没有机制。说了"会廉价"，没说清廉价怎么在她心里发生。
- **1分**：常识，口水。

### 维度四：场景度（15%）

- **10分**：场景细到监控感——时间、顺序、配文原句、那张照片怎么构图的。且是35+有事业男性的高配版行为，不是25岁小伙的行为。
- **8分**：场景和动作真实具体，但停在"很多人都这样"的通用级，没到"这就是我"。
- **6分**：有场景但是概括的（"发了很多照片"而不是"那张方向盘入镜的"）。
- **3分**：从概念出发往回编场景，或画像错位（把目标客群写成低龄行为）。
- **1分**：全程抽象概念，没有场景。

### 维度五：MAX贴合度（15%）

**V2.1定义修正**：贴合度考核的是**判断方式的继承，不是语气的模仿**。AI的目标不是变成MAX的复读机，而是被MAX世界观、审美方向、现实判断和表达边界校准过的大脑——像MAX会做出的判断，允许是MAX没说过的话。刻意堆口头禅反而扣分。

- **10分**：观察方式、判断方式、立场全在MAX体系内，是MAX会做出但还没做出的判断。换任何别的账号发，这条内容不成立。
- **8分**：判断方向和价值观对，个别用词或边界出了体系，小修即可。
- **6分**：内容本身不错，但是通用的——换任何情感号发都成立；或靠堆MAX口头禅制造表面相似。
- **3分**：违背校准库已有沉淀：用了硬禁用表达、空用慎用词、立场松动。
- **1分**：价值观跑偏——油腻、说教、讨好。（触红线不打分，走否决门。）

### 维度六：用户价值（20%）

- **10分**：看完必然带走东西——要么被刺穿（痛：这说的就是我），要么拿到可复用的判断（学：原来机制是这样），并且产生"这个人值得关注"的冲动。
- **8分**：有明确的一次刺痛或一次学到，但只有一次。
- **6分**：道理都对、说得也顺，但用户看完认知不变、行为不变。
- **3分**：说教感——用户感觉在被教育，不是被看见。
- **1分**：浪费用户三分钟。

## 3.3 评分流程（每篇必走）

1. **AI先自评**：交稿必须附六维分数、加权总分、自认最弱维度及原因、拿不准的1-2处。禁止只交稿不自评。
2. **人工后评**：你先打第一印象（直觉，不看AI自评），再打其余五维。最低要求：六维分数+最低分维度的一句话原因。
3. **算差值**：任何单维度差值≥2分，当场进校准库，写明AI高估/低估了什么。
4. **按总分走决策线**（见3.4）。

**AI自评封顶规则（V2.1新增，防自嗨）**：AI自评时以下情形强制封顶，不得高于上限——

|情形|封顶|
|---|---|
|增量检验答不出（说不出用户带走的新判断）|深度≤3|
|没有真实场景|场景度≤6|
|有场景但没有具体动作|场景度≤6，深度≤7|
|出现明显AI结构（排比连发、漂亮收尾、"你以为/其实"）|真实感≤6|
|只是概念解释，没有MAX式判断|MAX贴合度≤6|
|触红线|整篇0分，走否决门|

## 3.4 发布决策线（只看人工事前分）

|人工事前分|定级|动作|
|---|---|---|
|≥9.0|种子内容|优先拍摄，发布后纳入投放候选，表现好即追投|
|8.0–8.9|达标可拍|正常排期拍摄发布|
|7.0–7.9|改一轮|针对最低分维度修改一次，重评。二评仍不到8，毙|
|6.0–6.9|原则上毙|仅热点时效类内容可人工特批带病发布，并标记"特批"|
|<6.0|连选题毙|选题和角度一起重新判断|

**拍摄线是8.0，不是6.0。** 理由不变：内容的平均段位就是产品的隐形定价。账号被6分内容定义，30万的产品就撑不住。

# 第四部分：市场反馈分（第二、三层）

## 4.1 发布前先贴类型标签

每条内容发布前必须标记类型，因为不同类型对市场指标的期待不同，不标类型市场分会误导：

- **流量型**：负责拉新、扩大触达、制造第一层共鸣。考核互动分+客资。
- **人设型**（V2.1由"信任型"更名）：负责建立MAX本人的位置——MAX是谁、在什么圈层、为什么有资格判断贵气/审美/展示面/价值感、真实生活和审美能力是什么。允许互动平、客资少，考核完播和收藏。
- **转化型**：负责指向产品、服务、案例和咨询动机。考核咨询类评论和私信。

## 4.2 互动数据分（T+2、T+4、T+7快照，T+14收尾）

### 指标权重（按MAX项目定制）

|指标|权重|设计理由|
|---|---|---|
|完播率|25%|口播内容的生死线，决定平台推流；前3秒和节奏问题都暴露在这|
|评论率|20%|互动深度；其中"咨询意向评论"单独计入客资线索|
|收藏率|20%|用户认为有长期价值、会回看——最接近"学到了"的行为信号|
|转发率|15%|社交货币，内容替用户表达了他想说的|
|点赞率|10%|最廉价的动作，权重最低|
|主页访问率/关注转化率|10%|信任链路启动的信号|

### 率化计算（V2.1核心升级：考核内容接住了多少流量，不考核平台给了多少流量）

除完播率本身即是比率外，其余指标全部按每千次播放计算：

```
指标率 = 指标数 / 播放量 × 1000
```

绝对数高可能只是被推流了；率才反映内容质量。象限归类、金股判定一律用率化后的互动分。

**实施依赖（必须知道）**：播放量在抖音公开页不可见，只存在于创作者后台。因此播放量是人工每周录入项（与完播率一同录入，一周5条约1分钟）。播放量未录入时，互动分无法率化，按绝对数临时计算并标注"未率化-低置信"。

**最小播放量门槛**：播放量低于1000的视频不计互动分，标记"样本不足"。低播放下率指标全是噪声（500播放时3个赞和6个赞的"率"差一倍但毫无意义），不能进复盘。门槛值放配置文件，可调。

### 计分方式：和自己比，不和别人比

每个指标率对照**本账号近30天该指标率的中位数**计分：

|达到中位数的倍数|得分|
|---|---|
|<0.5倍|2分|
|0.5–1倍|4分|
|1–2倍|6分|
|2–5倍|8分|
|>5倍|10分|

**互动分 = Σ（指标分 × 权重），满分10分。**

**基线的渐进规则（V2.1明确）**：基线分三阶段演进——

1. 冷启动期（全账号样本<15条）：用配置文件里人工设定的固定基线。
2. 过渡期（全账号样本≥15条）：切换为全账号中位数。
3. 成熟期（某内容类型样本≥15条）：该类型切换为分类型中位数（流量型和人设型的合理互动结构本来就不同，混在一起比会误判人设型内容）。三个类型各自独立切换，不必等齐。

### 咨询意向评论的特殊处理

评论区出现"怎么找你""多少钱""有没有课""求联系"类评论，每条单独记录为**可归因客资线索**，挂到这条视频名下。这是单条视频唯一可靠的客资归因方式。

## 4.3 客资结果分（每周日结算，第三层）

**关键设计：客资分考核的是"本周内容组合"，不是单条视频。** 客资是账号级滞后指标，强行摊到单条头上会冤枉好内容。

计分对照**近4周滚动周均客资数**：

|本周客资 / 近4周周均|得分|含义|
|---|---|---|
|<50%|2分|本周内容组合失效，重点复盘|
|50%–80%|4分|偏弱|
|80%–120%|6分|正常波动|
|120%–200%|8分|本周打法有效，找出贡献源|
|>200%|10分|拆解本周全部要素，形成复制清单|

起步期没有4周基线时，由你设定周目标值代替周均，跑满4周后切换。

同时每周记录两个质量指标（不计分，但必须看）：

1. **客资质量**：本周客资中符合目标客群画像（30+、有付费能力）的比例。
2. **客资来源**：可归因到具体视频的客资，标注到对应视频，积累"哪类内容真正带客资"的样本。

# 第五部分：动态复盘机制（体系的发动机）

## 5.1 每周复盘的核心动作：把每条视频放进2×2象限

横轴是人工事前分（8为界），纵轴是互动分（6为界）：

### 象限一：事前高 + 市场高 → 金股

拆解可复制要素：选题类型、切入角度、前3秒钩子、增量判断是什么。进入复制清单，同类选题优先排产。

### 象限二：事前高 + 市场低 → 先查包装，再查判断

内容判断大概率没错，先排查三个外围因素：前3秒和标题钩子（看完播曲线在哪一秒掉的）、选题是否太小众、发布时间。**单条不调整评分标准**；同类内容连续3条落在这个象限，才说明我们的"好"和市场的"好"有系统性错位，进校准库。

### 象限三：事前低 + 市场高 → 最危险的象限，禁止直接庆祝

先问一个问题：**这条爆款伤不伤人设？** 如果是靠低段位钩子、擦边判断、降低语言段位换来的流量，定性为负资产，不复制，并检查它是怎么通过事前评分的。如果不伤人设、确实是我们看走眼了，说明评分体系漏掉了某个市场要素——这是最有价值的校准样本，重点分析。

### 象限四：事前低 + 市场低 → 体系正常

评分体系工作正常的证据，不需要动作。（特批发布的内容大多落在这里，用来验证特批是否值得。）

## 5.2 周复盘产出（固定三件事）

1. **象限表**：本周每条视频落位。
2. **差值记录**：AI自评vs人工的维度级差值≥2的条目；事前分vs市场表现的象限二、三条目。全部进校准库。
3. **下周排产判断**：基于金股清单和客资来源标注，定下周选题配比（流量型/信任型/转化型的比例）。

## 5.3 权重与标准的再校准节奏

- **每4周（或累计20条以上样本）**：检查一次六个事前维度与互动分的相关性——哪个维度的高分内容市场表现也好，哪个维度高分但市场无感。据此微调权重，升版本号。
- **每8周**：检查互动指标权重与客资的关系——是收藏高的内容带客资，还是评论高的带客资？据此调互动指标权重。
- **任何时候**：某个维度长期不区分好坏（所有内容都高分或都低分），说明锚点失效，重写锚点。

# 第六部分：常见问题预判

**问：事前分8.2但互动分只有3，这条内容算失败吗？** 不算失败，算样本。走象限二流程：先查钩子和完播曲线。内容判断和流量包装是两个问题，这套体系的价值就是能把它们分开。

**问：连续两周客资分都是2分，是不是该降低拍摄线多发内容？** 不是。先看客资质量和来源标注——大概率是选题配比问题（人设型发多了或流量型钩子弱了），不是数量问题。用降低质量换数量，违背核心原则三。

**问：AI自评和人工评分长期差值<1，能不能让AI独立初筛？** 能，这正是体系的目标状态。届时AI负责毙掉<7分的稿子，你只看7分以上的，校准成本降一个量级。但每月仍需抽查AI毙掉的稿子，防止AI过度保守杀掉好内容。
