# 飞书消息权限开通步骤

当前卡点：应用缺少发送消息权限。

报错需要的权限包括：

```text
im:message:send
im:message
im:message:send_as_bot
```

## 操作步骤

1. 双击：

```text
打开飞书消息权限页面.command
```

2. 在飞书开放平台页面里，申请或开通消息发送相关权限。

3. 检查应用是否启用了机器人能力：

```text
应用功能 / 机器人 / 启用机器人
```

4. 如果页面提示需要发布版本：

```text
创建版本 → 发布/启用应用
```

5. 回到本目录，重新双击：

```text
发送测试消息.command
```

## 成功标志

终端返回里出现：

```json
"code": 0
```

并且你的飞书收到测试消息。
