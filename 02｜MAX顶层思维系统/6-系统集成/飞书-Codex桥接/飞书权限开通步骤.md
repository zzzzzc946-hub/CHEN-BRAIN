# 飞书权限开通步骤

当前卡点：应用缺少通讯录权限。

报错需要的权限是：

```text
contact:user.id:readonly
```

这个权限用于：通过手机号或邮箱查询飞书用户的 `open_id` / `user_id`。

## 操作步骤

1. 双击 `打开飞书通讯录权限页面.command`。
2. 在打开的飞书开放平台页面里，申请或开通：

```text
contact:user.id:readonly
```

3. 如果页面提示需要发布版本：
   - 创建新版本
   - 提交发布
   - 启用/安装到当前企业

4. 完成后，回到本目录，重新双击：

```text
通过手机号查询飞书用户ID.command
```

5. 如果查询成功，复制返回里的：

```text
open_id: ou_xxxxx
```

6. 再双击：

```text
配置默认接收人.command
```

填写：

```text
接收人类型：open_id
接收人 ID：ou_xxxxx
```

7. 最后双击：

```text
发送测试消息.command
```

## 如果仍然失败

把终端窗口里的完整错误复制给 Codex。
