# 飞书机器人能力启用步骤

当前报错：

```text
Bot ability is not activated.
```

意思是：飞书应用还没有启用机器人能力。

## 操作步骤

1. 双击：

```text
打开飞书机器人能力页面.command
```

2. 进入飞书开放平台应用后台后，找类似入口：

```text
应用功能 / 机器人
```

或：

```text
功能管理 / 机器人 / 添加能力
```

3. 启用机器人能力。

4. 如果页面要求填写机器人信息，可以简单填写：

```text
机器人名称：Codex MAX 助手
描述：用于连接 Codex 与 MAX Obsidian 系统
```

5. 如果页面提示需要发布版本：

```text
创建版本 → 发布/启用应用
```

6. 回到本目录，重新双击：

```text
发送测试消息.command
```

## 成功标志

终端返回里出现：

```json
"code": 0
```

并且你的飞书收到测试消息。
